# Electron-quick-start

```bash
# Go into the repository
cd Electron-quick-start

#快速安装 Electron框架
npm config set electron_mirror http://npm.taobao.org/mirrors/electron/
npm config set electron_custom_dir "10.1.0"

# Install dependencies
npm install
# Run the app
npm start
```
